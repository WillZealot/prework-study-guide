Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Any code inside of the head is not visible to users.
  • HTML means Hyper-Text-Markup-Language.
  • HTML brings structure like a foundation or skeleton for the project.

CSS

  • CSS is what styles the page like adding colors, shapes, various fonts, and etc.
  • CSS coding is incorporated after an HTML skeleton is setup.
  • Be specific with CSS

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m
  • git add -a
  • git pull origin main
  • git push origin main
